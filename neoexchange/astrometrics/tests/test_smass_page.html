How to Use These Data

Data files:

  • = data values as ASCII text file. “Vis” and “NIR” refer to visible and near-IR wavelengths, respectively.
  • = graph of spectrum, .pdf format (requires Acrobat Reader.)

To download a data file, right-click on its icon in the table and choose “Save Link As...” or “Save Target As...”.
(Mac users with single-button mouse, use <CTRL>-click.)

“Last Updated” is the date that the spectrum was last added or most recently modified.

How to Cite These Data

To find the published or unpublished reference for a spectrum, click its “Data Reference” link.

Unpublished spectra made available here are considered to be in the public domain and may be freely used for any purpose. We welcome collaboration opportunities, but there is no pre-condition for collaboration for use of these data. (Requested citation and acknowledgement)
